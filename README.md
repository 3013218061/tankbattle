## 安装执行
1、执行maven命令clean package，可在target目录下生成jar包，直接执行jar包即可。<br/>
clean package<br/>
2、或者直接用eclipse导入源码，找到StartGame类，右键run as Java Application..如发现“找不到或无法加载主类”，请清理下工程。<br/>
